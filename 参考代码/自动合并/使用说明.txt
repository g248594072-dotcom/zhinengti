自动合并报表（SaleSmartly API 版）
================================

【使用步骤】
1. 解压文件夹到任意位置
2. 首次使用：复制 api-key.json.example 为 api-key.json，填入 API Token 和项目 ID
   （也可启动后在弹窗里配置 API）
3. 双击「自动合并报表.exe」（仅此一个程序文件，无需 _internal 文件夹）
4. 勾选访客标签（如 7..10）和接待客服
5. 点击「生成报表」
6. 在「输出」文件夹查看 Excel

【API 获取】
SaleSmartly → 项目设置 → 企业开发设置 → API Token
需要 Max 套餐。

【可选文件】
人员部门表.xlsx — 放在 exe 同目录，汇总表会显示正确部门。

【常见问题】
- 首次打开稍慢：单文件 exe 会解压到临时目录，第二次会快一些
- 再次打开仍慢：删除同目录 .cache 文件夹可强制刷新标签列表
- 启动失败：查看同目录 startup_error.log
- 拉数失败：查看 输出\error.log
- 关闭窗口即退出程序

【系统要求】
Windows 10/11 64 位，无需安装 Python。

【开发者重新打包】
在本目录双击运行 打包.bat
